# Parmida Mehrabi - Personal Portfolio

This is my personal portfolio website, built as the final project for Harvard University's CS50x 2025 course.

Live Site: https://parmidamehrabi.github.io

### Project Features
- Fully responsive design using pure HTML, CSS, and JavaScript
- Multi-language support: English • Persian (فارسی) • French (Français)
- Collapsible (accordion) sections using vanilla JavaScript
- Smooth scrolling navigation between sections
- Simple contact form
- Dark/Light theme toggle button using JavaScript
- Hosted for free on GitHub Pages

### Technologies Used
- HTML5
- CSS3
- JavaScript (no frameworks)
- GitHub Pages for hosting

### Website Sections
- About Me: Introduction and my passion for artificial intelligence
- Skills: Programming skills in C++, Python, HTML/CSS, and Java
- Projects: Showcase of completed projects (including this portfolio site)
- Goals: Short-term and long-term career objectives
- Contact: Ways to reach me (email and Instagram)
